# Parada Blog ⏤ Compartilhe Músicas em Comunidade

Esse projeto é inspirado na criação de Blogs Virtuais via [Blogger](https://www.blogger.com/about/?bpli=1), sendo um espaço virtual para visualização e criação de postagens entre pessoas, com um foco específico em compartilhamento de músicas e descrições sobre a mesma, ou uma opiniã sobre ela em um espaço só.
